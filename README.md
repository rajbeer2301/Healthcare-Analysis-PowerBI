# Healthcare Analysis Dashboard

## Project Overview
This project features an interactive **Healthcare Analysis Dashboard** built using **Power BI**.
It focuses on analyzing **member screening eligibility, coverage, and gaps** across multiple demographics.

## Key Metrics
- Total Members Eligible for Screening: 6,597
- Members Received Screening: 4,184 (63%)
- Members Not Received Screening: 2,413 (37%)

##  Regional Analysis Logic
The dataset did not include a direct region column.
To enable regional insights, the Region field was derived using business logic based on **Original Preferred Language**.
